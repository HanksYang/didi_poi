# DiDi POI 热点系统 - 可执行文件分发指南

## 📦 什么是这个文件夹?

这是 DiDi POI 热点系统的**独立可执行版本**，可以直接运行，无需安装 Python 或任何依赖！

## ⚡ 快速开始 (只需 3 步)

### macOS 用户：
```bash
# 方案 1：双击 "启动.sh" 脚本（推荐）
# 或
# 方案 2：在终端运行
chmod +x DiDi_POI热点系统
./DiDi_POI热点系统
```

### Windows 用户：
```
直接双击 "启动.bat" 文件即可
或
双击 "DiDi_POI热点系统.exe" 文件
```

### Linux 用户：
```bash
chmod +x DiDi_POI热点系统
./DiDi_POI热点系统
```

## 🎯 运行后会发生什么?

1. 程序启动，显示服务地址: `http://127.0.0.1:5000`
2. 浏览器自动打开 Web 界面
3. 如浏览器未自动打开，手动访问上面的地址

## 🧭 操作指南

### Tab 1️⃣ - 热力图上传
- 拖拽或选择热力图图像文件
- 系统 AI 自动识别地点和热度数据
- 结果保存到数据库

### Tab 2️⃣ - 路线规划
- 输入起点（如"三里屯"）
- 输入终点（如"国贸"）
- 选择城市（北京/上海/广州等）
- 点击"📍 规划路线"
- 系统返回 3 条最优路线，显示：
  - 热度总和
  - 距离
  - 时长
  - 接单评分
  - 途经点详情
  - 地图预览

### Tab 3️⃣ - 设置
- 设置高德地图 API Key
- 设置火山引擎 API Key
- 密钥自动保存到 `.env` 文件

## 🔑 首次配置

1. 点击"⚙️ 设置"标签
2. 输入您的高德地图 API Key
   - 获取地址: https://lbs.amap.com/
3. 输入火山引擎 API Key
   - 获取地址: https://www.volcengine.com/
4. 点击"💾 保存密钥"
5. 配置完成！

## 🆘 常见问题

### Q: 如何停止程序?
A: 在终端中按 `Ctrl+C`

### Q: 浏览器没有自动打开怎么办?
A: 手动访问 http://127.0.0.1:5000

### Q: macOS 显示"无法验证"怎么办?
A: 
```bash
# 第一次运行需要授权
sudo xattr -d com.apple.quarantine DiDi_POI热点系统
# 或在安全性设置中允许
```

### Q: Windows 防火墙阻止怎么办?
A: 在 Windows 防火墙中允许这个应用

### Q: API Key 保存位置在哪?
A: 同一文件夹中的 `.env` 文件

### Q: 能否在另一台电脑上运行?
A: 可以！这就是独立可执行文件的优势
   - 只需复制整个文件夹到目标电脑
   - 无需任何环境配置
   - 直接双击启动

## 🔄 数据存储

- 热力图数据: `data/traffic.db` (SQLite 数据库)
- POI 权重数据: `output/poi_heat_weights.json`
- 路线规划地图: `output/route_plan.html`
- API 配置: `.env` 文件

## 🌐 系统需求

- **macOS**: 10.13+
- **Windows**: Windows 7+
- **Linux**: 任何现代 Linux 发行版
- **网络**: 需要网络连接（调用 API）
- **磁盘**: 至少 100MB 可用空间

   - 不要分享您的 API Key 给他人

2. **数据安全**

3. **端口占用**
   - 程序使用本地 5000 端口
