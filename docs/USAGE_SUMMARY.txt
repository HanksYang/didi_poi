===============================================================================
🔍 工作原理
===============================================================================

1. 图像分析
   - 提取热力图中的红色像素分布
   - 计算热度强度特征

2. Claude Vision 分析
   - 将图像发送给 Claude API
   - Claude 识别地理位置和热度分布
   - 生成 100 个 POI 列表

3. 热度权重计算
   - 为每个 POI 计算热度分数 (0.0 - 1.0)
   - 按热度从高到低排序

4. 数据导出
   - 保存为 JSON 和 CSV 两种格式
   - 包含排名、名称、地区、类别、坐标、热度分数

===============================================================================
🔧 故障排查
===============================================================================

问题 1: "ANTHROPIC_API_KEY not found"
解决: 检查 .env 文件是否正确配置了 API Key

问题 2: "ModuleNotFoundError: No module named 'anthropic'"
解决: 运行 pip3 install anthropic --break-system-packages

问题 3: API 调用失败
解决: 检查网络连接和 API Key 是否有效

问题 4: JSON 解析失败
解决: Claude 响应格式可能不符合预期，检查图像质量

问题 5: 热度分数都是 0 或 1
解决: 检查热力图图像是否有明显的红色热度区域
