智能热力图识别 + 路线规划

📊

拖拽或点击上传热力图

支持 JPG / PNG / GIF 格式，最大 50MB

🗺️ 路线规划

输入起点和终点，系统自动规划最优路线

⚙️ 设置 API 密钥

📍 高德地图 API Key

用于地名解析和路线规划

🔗 申请 Key

点击申请高德地图 Web 服务 API → 创建应用 → 获取 Key

🔥 火山引擎 API Key

用于 AI 识别热力图

🔗 申请 Key

点击申请火山引擎 ARK API → 账户充值 → 创建 API Key

📖 使用说明

  1. 高德地图：用于将地名转换成坐标，以及规划路线
  2. 火山引擎：用于 AI 识别热力图中的地点和热度中心位置
  3. 两个 Key 都是必需的，系统才能完整工作
  4. 密钥会保存在本地 .env 文件中
  5. 首次使用前请先在"设置"标签页配置好 API Key
