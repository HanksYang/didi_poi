## 🔍 使用步骤

### 1️⃣ 启动应用
```bash
cd ~/didi_poi
pip3 install -r requirements.txt --break-system-packages
python3 run_app.py
```

### 2️⃣ 上传您的热力图（36dbc文件）
1. 在**📤 热力图上传**标签页
2. 拖拽或点击选择您的热力图文件

### 3️⃣ 先进行"图像诊断"（推荐）
1. 点击 **🔍 图像诊断** 按钮
2. 系统会自动分析：
   - ✓ 文件分辨率
   - ✓ 对比度
   - ✓ 热点区域清晰度
   - ✓ 文件大小

### 4️⃣ 查看诊断报告
诊断报告会显示：

| 项目 | 说明 |
|------|------|
| **红色分析** | 热点区域的红色像素比例 |
| **对比度分数** | 越高越好（应 > 50） |
| **分辨率** | 推荐至少 300x300 |
| **文件大小** | 推荐 < 10MB |

### 5️⃣ 根据建议改进（如需要）
诊断结果会给出具体建议：
- ⚠️ **问题列表** - 需要解决的问题
- 💡 **改进建议** - 如何改进

### 6️⃣ 确认无问题后提交识别
1. 点击 **🚀 提交识别**
2. 大模型会准确识别热点位置和地点

## 📊 诊断指标说明

### 热点像素比例 (Hot Pixels %)
- **< 0.5%** ⚠️ 热点不明显，可能识别困难
- **0.5% - 2%** ✓ 可以识别
- **> 2%** ✓ 热点明显，易于识别

### 对比度分数 (Contrast)
- **< 20** ❌ 图像模糊或褪色，需要改进
- **20-50** ⚠️ 一般，可以改进
- **> 50** ✅ 良好，有利于识别

### 分辨率 (Resolution)
- **< 300x300** ❌ 太低，建议使用高清截图
- **300x300 - 2000x2000** ✅ 合适
- **> 2000x2000** ✅ 很高，但可能文件过大

### 文件大小 (File Size)
- **< 1MB** ✅ 理想
- **1-10MB** ✓ 可接受
- **> 10MB** ⚠️ 需要压缩

## 💡 最佳实践

### 📸 获取最佳热力图
1. 使用**高德地图 App** 的热力图功能
2. 全屏显示，确保热点区域清晰
3. 在**良好光线**下截图或使用屏幕录制
4. 确保图片包含**道路名称**和**标志性地点**

### 🎨 优化热力图
如果诊断显示质量问题，可以：
1. 提高系统**对比度和亮度**设置
2. 使用**图片编辑工具**增强（如 Preview）
3. 确保截图时**没有反光或阴影**

### 📤 上传建议
1. 始终先运行 **🔍 图像诊断**
2. 解决所有 ⚠️ **问题**
3. 确认 ✅ **改进建议**全部通过
4. 再点击 **🚀 提交识别**

## 🔧 高级诊断（命令行）

如果您想在命令行中诊断文件：

```bash
# 创建诊断脚本
python3 << 'EOF'
from PIL import Image
import numpy as np

def diagnose(image_path):
    img = Image.open(image_path)
    arr = np.array(img)
    red = arr[:,:,0]
    hot = (red > 150).sum() / red.size * 100
    gray = np.mean(arr, axis=2)
    contrast = np.std(gray)
    
    print(f"分辨率: {img.size}")
    print(f"热点比例: {hot:.2f}%")
    print(f"对比度: {contrast:.2f}")
    print(f"文件大小: {os.path.getsize(image_path) / (1024*1024):.2f}MB")

diagnose("/path/to/your/heatmap.jpg")
EOF
```

## ❓ 常见问题

### Q: "热点区域不明显"是什么意思？
A: 图片中的红色区域（代表交通拥堵/接单热点）比例太低，大模型可能无法准确识别。建议使用清晰的高德地图热力图。

### Q: 如何改进对比度？
A: 
- macOS: 使用 Preview → Adjust → 提高 Contrast
- 或上传到在线工具进行增强

### Q: 诊断通过后识别还是失败？
A: 
1. 确保 API Key 已配置（在"⚙️ 设置"中）
2. 检查网络连接
3. 确认图片包含可识别的地点名称

### Q: 支持哪些格式？
A: JPG、PNG、GIF 均支持，推荐使用 JPG
